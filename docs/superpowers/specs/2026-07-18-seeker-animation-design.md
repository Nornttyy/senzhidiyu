# 《森之低语》寻音者动作设计文档——整图程序化动画

- 日期：2026-07-18
- 状态：设计已获用户确认，待实现
- 上游文档：`docs/GDD-whispers-of-the-woods.md`、`docs/superpowers/specs/2026-07-17-slice-a-night-forest-design.md`

## 1. 背景与选型

寻音者立绘已定稿（`assets/processed/seeker.png`，486×870 透明底）。切片 A 呈现方式已定为立牌立绘 + 水平翻转表朝向 + y 排序遮挡；按 48px/m 换算，角色屏上高度约 80–100px，且多数时间处于黑夜 + 光圈的低可见度环境。

据此选定**整图程序化动画**：不新增任何美术，全部动作由代码驱动这一张立绘的位移/旋转/缩放缓动。

已评估并否决的备选：

- **拆件提灯摆动**：把提灯抠成独立图层挂在手上做钟摆，光圈随灯位晃动。氛围加成大但需对立绘做修补手术，留作后续升级方向（见 §8），不阻塞本设计。
- **AI 逐帧序列帧**：千问出图帧间一致性差、每帧需过抠底管线并手动对齐，在 80px 显示尺寸下收益趋零。现阶段不做。

## 2. 范围

切片 A 需要的动作仅 4 个：**待机、行走、采集（砍木/挖矿共用）、放置**（放置无专门动作）。

不做：八方向朝向、侧面行走图、受击/战斗动作、逐帧序列、骨骼拆件。

## 3. 核心机制

- **锚点**：脚底中心（anchor 0.5, 1.0）。所有旋转与缩放以脚为轴，保证角色"踩在地上"而非悬空摇摆。
- **朝向**：`scale.x` 取 ±1 翻转，翻转带 0.1 秒缓冲，防止水平速度在零附近抖动时立牌频闪。
- **驱动方式**：动画是纯表现层。sim 层只暴露状态（`idle | walking | gathering`、朝向、`gatherProgress`），render 层的动画器读状态 + 插值时间计算 transform。sim 不知道动画存在，维持既定的 sim/render 分离（联机预留不受影响）。

## 4. 动作规格

### 4.1 待机

呼吸感：`scale.y` 在 1.000–1.015 间正弦起伏，周期 2.5 秒；`scale.x` 微量反向补偿保持体积感。提灯光晕闪烁复用切片 A 光照系统的火光呼吸，不在动画器内另做。

### 4.2 行走

- y 向颠簸：幅度 5px（渲染坐标），波形正弦取绝对值（只向上跳，落回地面）。
- 步频与移速挂钩：`步频 = 速度 / 步幅`，步幅 2m。默认移速 4 m/s 时即 2 步/秒。
- 身体前倾 4°（朝移动方向）。
- 颠簸波谷（"落脚"瞬间）触发落脚事件：脚下喷 1–2 粒尘土粒子 + 脚步声，使画面、粒子、音效对齐同一拍子。
- 停止移动时，0.15 秒内回弹至待机姿态。

### 4.3 采集（砍/挖共用）

以 1.2 秒（`config` 中的采集动作时长）为一个循环：

| 阶段 | 时间 | 表现 |
|---|---|---|
| 蓄力后仰 | 0 – 0.3s | 向后倾至 -8°，缓入 |
| 快速前劈 | 0.3 – 0.45s | 前倾至 +15°，缓出 |
| 命中瞬间 | t = 0.45s | 触发命中事件：目标实体抖动 + 资源粒子（低语木=金色萤火虫、萤石=蓝色碎屑）+ 敲击声 |
| 回正 | 0.45 – 1.2s | 缓动回到 0° |

采集进度反馈由循环本身承担（每循环 = 1 次采集判定），与切片 A "1.2 秒动作时间与进度反馈"的要求对应。角色朝向锁定为面向目标实体。

### 4.4 放置

角色保持待机动画；提灯柱预览半透明跟随（切片 A §4.4 已定），无专门动作。

## 5. 架构落点

- `src/render/characterAnimator.ts`：纯函数。输入 `(状态, 朝向, gatherProgress, 上一帧时间, 本帧时间)`，输出 `{x偏移, y偏移, 旋转, scaleX, scaleY}` 与本帧事件列表（落脚/命中）。事件按跨帧判定：事件时间点落在 `(上一帧时间, 本帧时间]` 区间内即触发一次，保证任意帧率下不丢帧、不重复。无内部可变状态，便于单测。
- 全部数值（振幅、频率、角度、时长、缓动参数）集中于 `src/config.ts`。
- 素材缺失时的程序绘制占位图形（切片 A 既定回退机制）同样被动画器驱动。

## 6. 数值初稿（config.ts，全部可调）

| 项 | 值 |
|---|---|
| 待机呼吸幅度 / 周期 | scale.y 1.000–1.015 / 2.5s |
| 行走颠簸幅度 / 步幅 | 5px / 2m（4 m/s 时 2 步/秒） |
| 行走前倾角 | 4° |
| 翻转防抖缓冲 | 0.1s |
| 停止回弹时长 | 0.15s |
| 采集蓄力 / 前劈 / 命中时刻 / 回正 | 0.3s / 0.15s / 0.45s / 至 1.2s |
| 采集后仰角 / 前劈角 | -8° / +15° |

## 7. 测试与验收

- vitest 单测（表现层纯函数可测部分）：同输入同输出的确定性；行走步频与落脚事件时序；采集命中事件恰在 0.45s 触发且每循环一次；翻转防抖（速度符号快速交替时不频闪）。
- 视觉验收：浏览器试玩，主观确认踩地感、动作与音效粒子的拍子对齐、黑夜环境下动作可读。

## 8. 后续升级方向（非承诺）

拆出提灯为独立图层做钟摆摆动，光圈锚定灯的实际位置随之晃动；需为立绘补全被灯遮挡的斗篷区域（一次性修补手术 + 管线扩展）。
